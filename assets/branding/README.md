# SuperLocalMemory brand assets

`slm-mark.svg` is the canonical transparent product mark. Use it at small
sizes and against either light or dark surfaces. The wordmark variants are for
horizontal lockups: `slm-wordmark-dark.svg` on dark surfaces and
`slm-wordmark-light.svg` on light surfaces.

The mark is deliberately a local memory core: a bounded ring, connected
evidence nodes, and a visible central retrieval point. Do not recolor, crop,
or place it on an untested busy background. Raster derivatives are generated
from these SVG masters; the SVGs remain the source of truth.
