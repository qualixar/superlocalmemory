# codex-plugin/ — SuperLocalMemory Codex Plugin

This directory is the **Codex plugin** for SuperLocalMemory v3.8.1.

It is maintained manually (not generated by `scripts/build-plugin.mjs`).

## Structure

```
codex-plugin/
  AGENTS.md              # Project rules — copy to your repo root as AGENTS.md
  README.md              # Install guide for Codex users
  _GENERATED.md          # This file
  .codex/
    config.toml          # MCP server registration (project-scoped)
  hooks/
    hooks.json           # Lifecycle hooks (session-level)
  scripts/
    slm-launch           # POSIX venv launcher (advanced use)
    slm-launch.bat       # Windows venv launcher (advanced use)
    ensure-venv.sh       # POSIX venv bootstrap utility
    ensure-venv.bat      # Windows venv bootstrap utility
  skills/
    slm-cache/SKILL.md
    slm-compress/SKILL.md
    slm-graph/SKILL.md
    slm-recall/SKILL.md
    slm-remember/SKILL.md
    slm-session/SKILL.md
    slm-status/SKILL.md
```

## Keeping in sync with plugin-src/

When releasing a new version of SuperLocalMemory:
1. Update `AGENTS.md` version footer to match `pyproject.toml` version.
2. Re-copy skill files from `plugin/skills/*/SKILL.md` (built by `npm run build:plugin`).
3. Run `~/.slm-venv/bin/python -m pytest tests/test_plugin/test_codex_plugin.py -q` to verify.

## Source of truth

| File                   | Source                               |
|------------------------|--------------------------------------|
| `AGENTS.md`            | `plugin-src/rules/AGENTS.md`         |
| `skills/*/SKILL.md`    | `plugin-src/skills/*/SKILL.md`       |
